# Movie Screening Schedule System

## Description
This project is a simple Java console application that allows users to manage a movie screening schedule.

## Features
- Add new movies
- View movie list
- Update movie information
- Delete movies
- Save data to file

## Technologies
- Java
- File I/O
- OOP principles

## OOP Used
- Encapsulation (private fields, getters/setters)
- Inheritance (Movie -> SpecialMovie)
- Polymorphism (method overriding)

## How to Run
1. Compile all files
2. Run Main.java
3. Follow menu instructions
